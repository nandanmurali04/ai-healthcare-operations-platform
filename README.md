# AI-Driven Healthcare Operations & Intelligence Platform

This project is a full-stack AI system designed to demonstrate how machine learning, data warehousing, and business intelligence can be used to improve hospital operations and decision-making.

The system simulates patient triage, appointment scheduling, and operational analytics using real-world healthcare data.

---

## Key Features

- AI-based patient triage using machine learning
- Urgency classification (Low / Medium / High)
- Department recommendation based on symptoms
- Appointment scheduling logic
- Healthcare data warehouse (star schema design)
- Business intelligence dashboards for operational insights

---

## Tech Stack

- Backend: FastAPI (Python)
- Machine Learning: Scikit-learn
- Frontend: React
- Database: PostgreSQL
- Data Warehouse: Star Schema
- BI & Analytics: Power BI / Apache Superset
